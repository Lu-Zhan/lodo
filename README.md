# lodo — 纠缠式提醒待办

lodo 是一个"不完成就不放过你"的待办 app:事项到期后可以**完成**或**稍等**,
忽略或稍等都会每隔一个稍等间隔(默认 15 分钟)**重复提醒,直到你完成为止**。
配合 AI 助手,一句话就能创建、修改、完成、删除待办。

| 平台 | 说明 |
|---|---|
| `ios/` | iPhone / iPad / Mac(SwiftUI,iOS 17+ / macOS 14+),**主力版本** |
| `android/` | Kotlin + Jetpack Compose(Android 8.0+),功能对齐 iOS |
| `web/` | Streamlit 演示版,仅作参考 |

---

## 快速上手

1. **添加事项**:点右下角 **+**
   - 上方 **AI 输入**:一句话描述,如"明天3点开会一小时",支持语音;解析结果自动填入下方表单,标"✨AI 已填写";没说时长时 AI 会按你的历史习惯建议(高亮显示,可改)
   - 下方 **手动表单**:内容、日期/时间、全天、每天/每周重复(可多时间点)、时长
2. **等提醒来**:到点后收到通知;不理它?稍等间隔后**再次提醒**,直到完成
3. **响应提醒**(通知按钮或主页到期卡):
   - **完成 / 开始了**——有时长的事项分两段:先提醒"该开始了",到点再问"完成了吗"
   - **稍等 N 分钟**——过一会儿再催
   - **改期**(到期卡右上角)——AI 给出"今晚 20:00 / 明早 9:00 / 周六上午"等候选,一点即改

## 主页

- **日期条**:横滑选日期,默认今天;下方按"今天待办 / 未来待办"分组
- **到期提醒卡**:所有已到期的事项集中在顶部,带操作按钮
- **滑动手势**(可在设置关振动):
  - 待办行:**右滑 = 完成**,**左滑到底 = 删除**,点击 = 编辑
  - 已完成页:**右滑 = 恢复为未完成**,左滑 = 删除
- **下拉列表**(iOS)/ 顶栏 **✨**(Android):唤出**全局 AI 助手**

## 全局 AI 助手

大输入框 + 语音按钮(讲完自动提交),一句话什么都能干:

| 你说 | 它做 |
|---|---|
| "明天上午开会,下午取快递,周五交报告" | 一次创建 3 条(先给确认清单) |
| "把开会改到晚上8点" | 直接打开该事项的编辑页并改好 |
| "开会完成了,把取快递删了" | 完成 + 删除(过确认清单,不会静默执行) |
| "提醒我交材料"(没说时间) | 反问"什么时候?"并给出候选时间,点一下补全 |

## 每日汇总

设置里开启后,在指定时间推送今天开始/到期的事项汇总——**正文由 AI 一句话概括,突出重点**。
支持一天多个时间点、每天或每周(可选星期几,如工作日)。

## AI 越用越懂你

- **时长记忆**:完成有时长的事项后,AI 归纳"会议 ≈ 60 分钟"这类规律,下次创建没说时长时自动建议;偶尔会轻量询问"实际用了多久?"(同类事项时间稳定后不再问);记忆内容可在设置里查看、编辑、重置
- **本周洞察**:已完成页顶部每周一句正向回顾(如"这周完成 12 件,比上周多 3 件"),只肯定进步、绝不打分施压;不想看可关闭
- **AI 个性**:默认公事公办;可选"高效秘书 / 温柔陪伴 / 严格教练 / 幽默轻松",或自定义说话风格——只影响反问、汇总、洞察的语气,不影响解析

## Siri 与小组件(iOS)

- 对 Siri 说:**"在 lodo 里添加事项"**(口述内容,AI 解析直接建好)、**"lodo 今天有什么安排"**、**"在 lodo 里完成事项"**、**"打开 lodo 助手"**;四个动作也可在「快捷指令」里编排自动化
- **桌面小组件**:显示即将到来的事项,右侧 **+** 一键进入添加页

## AI 服务配置

设置 → AI 服务:

- **默认 DeepSeek**,也支持 OpenAI、通义千问、Kimi、智谱,或自定义任何 OpenAI 兼容接口(如 OpenRouter、本地 Ollama);填对应服务商的 API key 即可,key 加密保存在本机(iOS 钥匙串 / Android Keystore),按服务商分开记
- **苹果智能**(iOS 26+,iPhone 15 Pro 及以上):免 key、离线、数据不出设备,选中即用

> 隐私说明:使用云服务商时,你输入的自然语言和当前待办列表会发送给所选 AI 服务解析;
> 选苹果智能则全部在设备端完成。

## 开发者:构建与测试

```bash
# iOS / macOS(需要 Xcode 26)
open ios/Lodo.xcodeproj
cd ios/LodoCore && swift test          # 调度器核心逻辑测试

# Android(JDK 17)
cd android && ./gradlew :app:assembleDebug
./gradlew :app:testDebugUnitTest

# Web 演示
cd web && pip install -r requirements.txt && streamlit run app.py
```

三端共享同一套调度语义(纠缠提醒、两阶段时长、重复规则),详见 `CLAUDE.md`。
